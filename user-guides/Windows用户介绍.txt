🪟 SmartEmailSender for Windows - 智能邮件批量发送工具

亲爱的Windows用户，

我为您推荐一款专业的邮件批量发送工具 - SmartEmailSender！这是一款完美集成Microsoft 365的智能邮件发送应用，让您轻松实现个性化邮件的批量发送。

🌟 主要功能：
✅ Excel数据导入 - 直接使用Excel文件作为邮件数据源，支持个性化变量
✅ Microsoft 365群组集成 - 支持发送到群组或群组成员个人邮箱
✅ 富文本HTML编辑器 - 专业的邮件编辑，支持格式设置和个性化内容
✅ 智能附件管理 - 支持通用附件和个性化附件（按姓名自动匹配）
✅ 多种发送模式 - 测试、草稿、正式发送，确保邮件准确无误
✅ 安全可靠 - Microsoft官方API认证，本地token存储

🎯 适用场景：
• 企业通知和公告批量发送
• 员工培训邀请和证书发放
• 客户关怀和营销邮件
• 学生成绩单和通知发送

🖥️ 系统要求：
• Windows 10 或更高版本（64位）
• Microsoft 365工作/学校账户
• 稳定网络连接
• Microsoft Edge WebView2（自动检测安装）

📦 获取方式：
我将发送给您一个自动构建包：SmartEmailSender_Windows_Complete_Builder.zip (1.6MB)

🔧 构建安装步骤：
1. 解压SmartEmailSender_Windows_Complete_Builder.zip
2. 双击运行"双击开始构建.bat"
3. 构建器将自动：
   ✅ 检查和安装Python环境
   ✅ 下载安装所有依赖包
   ✅ 构建Windows应用程序
   ✅ 创建专业安装程序
   ✅ 运行安全检查
4. 构建完成后在releases文件夹获取：
   📦 SmartEmailSender-Windows.zip (绿色版，约400-600MB)
   🔧 SmartEmailSender-Setup.exe (专业安装程序)

⏱️ 构建时间：首次约5-15分钟（需下载约200MB依赖）

🚀 使用流程：
1️⃣ 启动应用，使用Microsoft 365账户登录
2️⃣ 导入Excel数据或选择Microsoft 365群组
3️⃣ 编写邮件内容，插入个性化变量（如{{姓名}}、{{公司}}）
4️⃣ 添加附件（支持个性化附件匹配）
5️⃣ 测试发送给自己验证效果
6️⃣ 批量发送，实时查看进度

💡 贴心提示：
• 构建过程中如有杀毒软件提示，请选择"允许"
• 首次使用建议先发送10-20封测试邮件
• 个性化附件命名格式：姓名_文件名.pdf（如：张三_证书.pdf）
• 支持变量：{{姓名}}、{{邮箱}}、{{公司}}、{{职位}}等
• Microsoft 365限制：每分钟30封，每天10,000封

🛠️ 故障排除：
如遇到问题：
• 确保网络连接正常
• 运行时选择"以管理员身份运行"
• 关闭杀毒软件重试
• 确保有足够磁盘空间（至少2GB）
